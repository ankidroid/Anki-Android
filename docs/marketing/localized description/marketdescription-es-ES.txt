¡Memoriza cualquier cosa con AnkiDroid!

AnkiDroid te permite aprender tarjetas de manera eficiente, mostrándotelas justo antes de que se te olviden. Es totalmente compatible con el software de repaso espaciado Anki (incluyendo sincronización), que está disponible para Windows, Linux y MacOS.

Estudia todo tipo de cosas donde y cuando quieras. ¡Aprovecha los tiempos muertos en los viajes en autobús, en las colas de los supermercados o en cualquier otra situación de espera!

Añadir material a través de la aplicación de escritorio Anki o directamente mediante Ankidroid.

Crea tus propios mazos de tarjetas o descarga mazos gratuitos creados para muchos idiomas y temas (hay más de 6.000 disponibles). ¡La aplicación soporta incluso la creación de material de forma automática a partir de un diccionario!

★ Características principales:
• contenidos de tarjeta soportados: texto, imágenes, sonidos
• repaso espaciado (algoritmo de SuperMemo 2)
• integración de texto a voz
• más de 6000 mazos prediseñados
• dos widgets
• estadísticas detalladas
• sincronización con AnkiWeb
• código abierto

★ Características adicionales:
• escribir las respuestas (opcional)
• pizarra
• explorador de tarjetas
• editor/creador de tarjetas
• agregar tarjetas instantáneamente a partir de otras aplicaciones como diccionarios
• integración con diccionarios (ColorDict, GoldenDict, Leo.org, Aedict, diversos diccionarios online)
• soporte para fuentes personalizadas
• completo sistema de copias de seguridad
• navegación por gestos, toques, agitado
• totalmente personalizable
• modo empollar
• limita el estudio por etiquetas, tiempo o tarjetas
• modo noche
• soporte para zeemote
• tres temas de interfaz diferentes
• traducido en 23 idiomas
