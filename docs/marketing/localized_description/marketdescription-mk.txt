Запомни нешто со AnkiDroid!

AnkiDroid те овозможува да ги научиш флеш картичките многу ефикасно и прикажувајќи ги пред да заборавите. Целосно е компатибилен со проредниот софтвер за повторување Anki (вклучувајќи синхронизација) кој е достапен за Windows, Linux и MacOS.

Проучи ги сите видови на работи каде и кога сакаш. Направете добра употреба на мирни времиња на автобуски патувања, во редици во супермаркет или во каква било друга ситуација на чекање!

Направете свој флеш-палуби или преземете бесплатни палуби, составени за многу јазици и теми (повеќе од 6000 достапни).

Додадете материјал преку десктоп апликацијата Anki или директно преку AnkiDroid. Апликацијата дури и поддржува додавање на материјал автоматски од речникот!

★ Клучни карактеристики:
• Поддржен содржина на картичката: текст, слики, звуци, LaTeX
• распоредено повторување (супермоно алгоритам 2)
• Интегрирање текст-во-говор
• повеќе од 6000 подвижни палуби
• додаток за напредок
• детални статистики
• синхронизирање со AnkiWeb
• отворен извор

★ Дополнителни функции:
• пишува одговори (опционално)
• whiteboard
• card editor/adder
• card browser
• tablet layout
• import existing collection files (via Anki Desktop)
• dictionary integration (ColorDict, GoldenDict, Leo.org, Aedict, diverse web dictionaries)
• add cards by intent from other applications like dictionaries
• custom font support
• full backup system
• navigation by swipe, tap
• customisable
• dynamic deck handling
• night mode
• 27 localisations
• All previous AnkiDroid versions can be downloaded from the website
