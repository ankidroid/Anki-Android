Parkerigu ion ajn per AnkiDroid!

AnkiDroid ebligas al vi efike memorigi fulmokartojn per montri ilin tuj antaŭ forgeso. Ĝi estas plene kongrua kun komputila programo Anki (inkluzive samtempigadon), kiu estas disponebla por operaciumoj Windows, Linukso kaj MacOS.

Memorigu kion ajn kaj kie ajn vi volas. Efike uzu liberan tempetojn dum vojaĝi aŭtobuse, en superbazara vico kaj ĉiam, kiam vi perdas tempon atendante!

Kreu viajn proprajn fulmokartojn aŭ senpage alŝutu kartarojn pri multaj lingvoj kaj temoj (pli ol 6000 disponeblaj).

Aldonu lern-enhavon per komputila programo Anki aŭ senpere per AnkiDroid. La aplikaĵo eĉ subtenas aŭtomate aldoni materialon el vortaro!

★ Ĉefaj eblaĵoj:
• subtenata enhavo por kartoj: teksto; bildoj, sonoj, LateX
• plani ripetojn laŭ algoritmo “supermemo 2”
• tekst-al-voĉo
• pli ol 6000 antaŭ-kreitaj kartaroj
• progresa fenestraĵo
• detala statistiko
• samtempigado kun AnkiWeb
• malfermkoda programaro
