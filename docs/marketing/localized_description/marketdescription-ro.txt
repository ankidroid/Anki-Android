 Caracteristici principale: 
 - conținutul suportat: text, imagine, sunete, Latex 
 - repetiție spațiată (algoritmul supermemo 2) 
 - integrare text-to-speech 
 - >6000 pachete deja făcute 
 - widget ce indică progresul și timpul rămas 
 - statistici detaliate 
 - sincronizare cu AnkiWeb 
 - open source 
 
 Caracteristici adiționale: 
 - scrie răspunsurile (opțional) 
 - whiteboard 
 - editor de carduri 
 - card browser 
 - tablet layout 
 - importă fișiere din Anki Desktop 
 - integrare cu diverse dicționare web (ColorDict, GoldenDict, Leo.org, Aedict etc.) 
 - adaugă carduri din alte aplicații precum dicționare 
 - suport pentru font personalizat 
 - sistem full backup 
 - navighează prin deplasare cu degetul, apăsare, înclinare 
 - complet personalizabil 
 - control dinamic al pachetelor 
 - mod de noapte 
 - 27 de localizări
